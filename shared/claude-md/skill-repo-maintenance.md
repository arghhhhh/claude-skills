## Skill Repo Maintenance

When editing, updating, or adding skills/agents, read `~/.claude/skills/skill-repo-maintenance/SKILL.md` first. It ensures changes are versioned, committed, pushed, and synced across machines.

**Always pull before editing skills** — another machine may have pushed changes.

Trigger phrases: "update skill", "add skill", "edit skill", "new skill group", "sync skills", "skill version", "bump version"
