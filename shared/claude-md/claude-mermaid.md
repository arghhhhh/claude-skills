## Claude Mermaid - Diagram Rendering with Live Preview

When the user wants to create, render, or preview a Mermaid diagram — flowcharts, node graphs, sequence/state/class/ER diagrams, mind maps, Gantt charts, or any architecture/data-flow diagram — read `~/.claude/skills/claude-mermaid/SKILL.md`.

The `claude-mermaid` MCP server provides two tools — `mermaid_preview` (live-reload browser preview) and `mermaid_save` (export to SVG/PNG/PDF). If the tools are missing, the MCP server is not registered or Claude Code needs a restart — see the skill file.

Trigger phrases: "mermaid", "diagram", "flowchart", "flow chart", "node graph diagram", "sequence diagram", "state diagram", "class diagram", "ER diagram", "graph TD", "graph LR", "architecture diagram", "render diagram", "draw a diagram"
