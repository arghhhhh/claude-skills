## App UI - Unity UI Framework

When working with Unity App UI (UI Toolkit components, MVVM, navigation, Redux, or theming), invoke the matching app-ui skill before writing code:

| Skill | Use When |
|-------|----------|
| `app-ui` | General App UI components, UXML setup, Panel configuration |
| `app-ui-mvvm` | MVVM pattern, ObservableObject, RelayCommand, AppBuilder, dependency injection |
| `app-ui-theming` | USS variables, custom themes, dark/light mode, scale factors, BEM conventions |
| `app-ui-navigation` | NavGraph, NavHost, NavController, destinations, AppBar, Drawer, BottomNavBar |
| `app-ui-redux` | Store, Slices, Reducers, Actions, AsyncThunks, middleware, Redux DevTools |

Requires `com.unity.dt.app-ui` package in the Unity project.

Trigger phrases: "app ui", "UXML", "panel", "MVVM", "observable", "redux", "store", "slice", "navgraph", "navhost", "theme", "USS variables", "BEM"
