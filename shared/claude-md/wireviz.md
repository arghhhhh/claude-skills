## WireViz - Wiring & Cable Harness Diagrams

When user wants to document or diagram wiring — cable harnesses, connector pinouts, wire colors, point-to-point connections between boards/sensors/connectors — read `~/.claude/skills/wireviz/SKILL.md`.

Renders plain-text YAML to SVG/PNG/HTML + BOM via the `wireviz` CLI (requires Graphviz `dot` on PATH).
