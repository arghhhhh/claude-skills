## Linear CLI - Linear.app from the Terminal

When the user wants to work with Linear.app — view/create/update/start issues, search issues, manage projects, milestones, cycles, initiatives, labels, documents, or teams — read `~/.claude/skills/linear-cli/SKILL.md`.

The `linear` CLI (schpet/linear-cli) is git/jj-aware and infers the issue from the current branch; most read commands support `--json` for agent workflows. Requires `linear auth login` with an API key.

Trigger phrases: "linear", "linear.app", "linear issue", "ENG-123" (any TEAM-123 issue key), "start issue", "create issue", "linear project", "milestone", "cycle", "initiative", "linear doc"
