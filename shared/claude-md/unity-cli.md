## Unity CLI - Unity Editor Automation

For any Unity Editor automation — scene inspection, GameObject editing, prefab workflows, C# code navigation/editing, asset management, input system, testing, or UI automation — use the `unity` agent or invoke the matching unity-cli skill.

The `unity-cli` binary must be installed and the `com.akiojin.unity-cli-bridge` UPM package must be in the Unity project.

Trigger phrases: "unity", "scene", "gameobject", "prefab", "component", "inspector"
